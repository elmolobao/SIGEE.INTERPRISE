# SIGEE Enterprise 2.0 - Entrega 5C

## Módulo
Dashboard

## Arquivo para substituir
- `frontend/js/dashboard.js`

## Testes rápidos
1. Login como Master/SEC.
2. Verificar Dashboard Global.
3. Selecionar um NTE no filtro do Dashboard.
4. Verificar se cards de Escolas, Acervos, Municípios e Processos atualizam.
5. Login como Técnico/Administrador e confirmar Dashboard restrito ao NTE do usuário.

## Observação
Esta entrega não altera banco de dados nem `app.js`. O módulo faz override seguro das funções do Dashboard após o carregamento do sistema.
