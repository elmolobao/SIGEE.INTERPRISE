# SIGEE Enterprise 2.0 - Parte 4

## Módulos entregues

- `frontend/js/processos.js`
- `frontend/js/fluxo-operacional.js`

## Objetivo

Centralizar a listagem dos processos, contadores, ações do Master/SEC e transições do fluxo operacional, sem remover ainda o código legado do `app.js`.

## Como aplicar

Substitua no GitHub:

```text
frontend/js/processos.js
frontend/js/fluxo-operacional.js
```

Depois faça o commit e aguarde o Vercel publicar.

## Observação

Esta etapa é incremental: os módulos são aplicados por cima do sistema atual após o carregamento da página, preservando as funcionalidades consolidadas.
